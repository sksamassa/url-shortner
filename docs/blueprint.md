# **App Name**: LinkWise

## Core Features:

- URL Shortening: Accepts a long URL and generates a unique, shorter URL.
- Custom Alias: Allows users to create custom aliases for their shortened URLs.
- Analytics Dashboard: Displays key metrics such as click count, geography, device types, and referral sources. Supports data filtering and export.
- QR Code Generation: Automatically generate QR codes for each shortened URL. Offers options for customization (color and logo).
- Bulk URL Shortening: Allows users to upload a CSV file containing multiple URLs for bulk processing and shortening.
- Link Categorization: AI tool analyzes the destination web page, automatically suggesting one or more appropriate descriptive categories that the user can quickly accept to associate with a shortened link. The AI considers the content, metadata and other information to make the decision.
- Rate Limiting: Limit the number of requests from a single IP address or user to prevent abuse.

## Style Guidelines:

- Primary color: Dark purple (#6750A4) to convey sophistication and trust.
- Background color: Light gray (#F2F0F7) to provide a clean and professional look.
- Accent color: Teal (#00A3AD) for highlighting key interactive elements such as buttons and links.
- Body and headline font: 'Inter' sans-serif for a modern, neutral, and readable feel.
- Code Font: 'Source Code Pro' for displaying API keys, code snippets, and short codes.
- Use minimalist and clear icons to represent different features and metrics in the dashboard.
- Subtle animations for loading states and transitions to improve user experience.
